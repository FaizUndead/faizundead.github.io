How to play:

Memory is a counter game where the object is to find pairs.

When the game begins, all pictures are hidden

To Play:

  1. Select two cards to try to match the picture
  2. If you match the pictures cards disappear
  3. Go on until there will be no cards
  4. Good Luck! Have Fun!
